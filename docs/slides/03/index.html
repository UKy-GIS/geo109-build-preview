Map rates, not totals

  • Larger countries can have more people.
  • We normalize by the area of each country.
    • We create a population density map

$area / 1000000

Calculates area in square kilometers

"POP_EST" / ($area / 1000000)

Calculates people per square kilometer

Mapping other variables

  • Inspect us_census_counties_metadata.txt
    • They tell us column names
  • For example, if you wanted to map the median age of the population, you could use the following value:
    • SE_A01004_001
    • Note that the column starts with "SE_"
