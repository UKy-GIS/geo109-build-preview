Language of the Web

  • A web page uses three coding languages:
    • HTML
    • CSS
    • JavaScript
  • All are living standards.

HTML

  • HyperText Markup Language (link)
  • Began in 1993
  • Defines the structure of the web page.

CSS

  • Cascading Style Sheets (link)
  • Began in 1994
  • Defines the style of the web page.

JavaScript

  • The programming language of he browser (link)
  • Began in 1995
  • Defines the behavior of the web page.
  • Most popular language.

Mapbox

Mapbox

  • Primo 🍭 online map builder and hosting service.
  • Their Mapbox Studio has been compared to the PhotoShop of online maps.
  • Free to use, until your project gets very popular.

Get an account

  • at Mapbox.com.
  • Then go to Mapbox Studio
  • Mapbox Studio works entirely in the browser
  • and has excellent documentation.

Vector Tiles

  • The server sends geometry and attribute data to the client (you)
  • as points, lines, and polygons.
  • The browser renders the map on-the-fly using video game technology.

Studio's three sections

  • Dataset: vector data that can be edited
  • Tilesets: compiled datasets ready for styling
  • Styles: the map as a collection of tilesets
