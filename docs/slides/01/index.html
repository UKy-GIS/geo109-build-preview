Comma-separated values

  • CSV is a delimited text file
    • delimited: having fixed boundaries.
    • text: readable by humans.
    • file: data stored in a shareable document.
  • A common way to share data.

CSV format

  • First row of file is attribute field or name.
  • Each subsequent row is a data record.
  • Each record consists of one or more fields, separated by commas.

CSV field values

  • Text values are enclosed in double-quotes "
    • For example, "spice.stop.supper"
  • Number values are not enclosed in double-quotes
    • For example, -85.7617613
